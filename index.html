오늘의 실습 목표

  1. 페이지 이동에 따른 page_view 흐름 확인
  2. 상품 클릭(select_item) / 상세조회(view_item) 확인
  3. 장바구니 담기(add_to_cart), 체크아웃(begin_checkout), 구매(purchase) 확인
  4. 회원(member) vs 비회원(guest) 세그먼트 비교
